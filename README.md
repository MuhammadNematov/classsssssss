## 📖 О проекте

Корпоративный сайт компании **РСЭМ**, специализирующейся на электромонтажных работах и производстве щитового оборудования. Сайт представляет компанию, показывает ключевые показатели (10 лет опыта, 200+ проектов), рассказывает о корпоративной ответственности, демонстрирует сертификаты, лицензии, клиентов и реализованные проекты. Включает формы связи и призыв к диалогу с экспертом.

**Цель:** помочь потенциальным заказчикам быстро узнать о компетенциях компании и оставить заявку.

## ✨ Особенности

- Полностью адаптивная вёрстка (mobile first)
- Семантический HTML5 + Tailwind CSS
- Секции:
  - «Поможем запустить ваш проект» (hero с формой)
  - Блок с цифрами: 10 лет, 200+ проектов
  - Корпоративная ответственность
  - Сертификаты и лицензии (горизонтальный скролл)
  - Среди наших клиентов (слайдер / галерея)
  - Щитовое оборудование
  - Реализованные проекты
  - Хотите поговорить с экспертом? (фоновая картинка + кнопка)
  - Контакты + контактная информация
- Форма обратной связи (имя, телефон)
- Чистый CSS без лишних зависимостей

## 🛠 Технологии

- HTML5
- Tailwind CSS (CDN / локальная сборка)
- JavaScript (для интерактива)
- Git
